# Image tiler

## Prerequisites
 - node > 10.0.0
 - npm > 6.0.0
 - make sure you have run image tiler to generate image tiles
 - to verify image tiles please check [src/assets/images](src/assets/images) folder

## Installation
```
$ npm ci
```

## Running
Launch website

```
$ npm start
```

then open `http://localhost:3000/` in chrome
